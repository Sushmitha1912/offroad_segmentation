# Offroad Semantic Scene Segmentation

## Overview

This repository contains my implementation of the Duality AI Offroad Semantic Scene Segmentation Challenge.

The goal of this project is to train and evaluate a semantic segmentation model capable of identifying different terrain and environmental classes in off-road desert environments using synthetic data generated through Duality AI's Falcon platform.

## Dataset

The dataset used in this project was provided by Duality AI as part of the Offroad Semantic Scene Segmentation Hackathon.

The dataset includes:

* RGB color images
* Semantic segmentation masks
* Training dataset
* Validation dataset
* Test images for evaluation

Due to dataset size limitations, the complete dataset is not included in this repository.

## Performance Evaluation

Model performance was evaluated using the Intersection over Union (IoU) metric, which is the primary evaluation metric for semantic segmentation tasks.

### Results

* Mean IoU Score: **0.092 (9.2%)**

Sample segmentation outputs are included in this repository for qualitative evaluation.
